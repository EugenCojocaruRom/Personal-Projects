18FEB - Persons registry
Create a class that can hold the details of some registered persons, and then allow us to search for them, and get some statistical info about them.

1. Create a Person class having these properties:

 - cnp - an integer (optional: validate value is > 0)

 - name - a string (optional: validate it's not empty)

 - age - an integer (optional: validate value is > 0)

 - hasVoted - a boolean, true if person has voted in last elections

2\. You are given a PersonsRegistry interface. Create a PersonsRegistryImpl class that implements this interface.
